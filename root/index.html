 Blockchain

Here are a couple projects that I'm following:

Polkadot

Polkadot is a blockchain interoperability protocol created by the Web3 Foundation. It allows disparate blockchains to exchange data trustlessly and can secure multiple chains simultaneously with pooled security. You can find out more about how it works here.

MakerDAO

MakerDAO has created both a cryptocurrency (Dai) and a protocol by which Dai can maintain a stable value. Unlike other stablecoins, Dai's price volatility is mitigated in a decentralized way. You can find out more about how it works here.
